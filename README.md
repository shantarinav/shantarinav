## Главный пример: Рапорт

**Рапорт** — линейка легких локальных аналитических инструментов для Excel/CSV-выгрузок промышленного предприятия.

**Слоган:** Excel докладывает главное.

Идея Рапорта: не заменять корпоративную аналитическую платформу, а быстро проверять управленческие сценарии до тяжелой автоматизации. Пользователь загружает отчет, видит ключевые показатели, отклонения, зоны внимания и может оформить A3-разбор проблемы.

Что показывает этот проект:

- работу с реальной управленческой болью: Excel-выгрузки есть, но управленческое действие часто не фиксируется;
- осознанное ограничение: базовое приложение работает без обязательной серверной части;
- безопасный подход к данным: исходные файлы и сырые строки не сохраняются;
- локальные A3-разборы отклонений с ответственными, сроками и статусами;
- необязательный ИИ-сервис для подсказок и уточнения спорных случаев;
- документацию как часть продукта: архитектура, журнал решений, план развития, выпуски и демонстрационные данные.
