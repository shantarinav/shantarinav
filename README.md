## Продуктовые линейки

### Рапорт
**Рапорт** — линейка легких аналитических инструментов для руководителей и специалистов промышленного предприятия.
**Слоган:** Excel докладывает главное.

| Проект | Назначение | Статус |
|---|---|---|
| Рапорт: качество ССЗ | Контроль качества оформления сменно-суточных заданий | v1.2 |
| Рапорт: исполнительская дисциплина | Анализ сроков согласования и исполнения задач в СЭД Tessa | v0.9 |
| Рапорт: контроль печати | Анализ объемов печати, нагрузки на устройства и точек оптимизации затрат | v1.1 |
| Рапорт: SLA техподдержки | Анализ закрытых заявок, исполнения SLA и качества работы техподдержки | v1.1 |

Текущий технологический стек:

- React;
- TypeScript;
- Vite;
- Tailwind CSS;
- shadcn/ui primitives;
- lucide-react;
- IndexedDB для локальной истории KPI;
- xlsx для чтения Excel-файлов.

Архитектурный подход:

- единая SPA-линейка дашбордов;
- hash-routing для публикации в любой папке сайта;
- главная страница как единая точка загрузки файлов;
- определение типа отчета по структуре и содержимому файла, а не по имени;
- разделение бизнес-логики и React-компонентов;
- переиспользуемые UI-компоненты;
- единый визуальный стиль для всех дашбордов;
- локальное хранение только агрегированных KPI-снимков, без сохранения сырых строк отчетов.

Репозиторий: `raport-product-line`.
